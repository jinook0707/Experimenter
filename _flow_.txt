Below shows a process of an experiment script, generated by e_gui.py.
(2020.June.14)


class Experimenter
__init__()

  get_subject_info()
    SubjectInfoDialog class

  init_expmt()
    initialize variables and stimuli info setup

  start_session()

  start_trial()

  schedule_trial_procedure()

  present_img_setup()/ present_snd_setup()/ present_mov_setup()/ present_txt_setup()

  schedule_post_stim_presentation()
  
  chk_end_stim()
    if there's no more stimulus to present: end_stim_presentation()

  end_stim_presentation()
    schedule user-response-start, reaction-time-start and timeout

  ... (waiting for user's mouse-click on image or text; 
    keyboard input is also possible, but additional programming is required.)

  onClick()

  end_trial()
    stop timers, remove all stimuli

    determine_trial_correctness()

    write_response()

    determine_trial_feedback()
      present_feedback()
        if negative: change_bg_color()

    if there're more trials: go back to start_trial()
  
  end_session()

quit program.
